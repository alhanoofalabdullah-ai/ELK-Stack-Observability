# Project 01 – ELK Stack Fundamentals

Introduction to ELK Stack (Elasticsearch, Logstash, Kibana) for centralized logging and observability.

---

# Overview

This project demonstrates the basic architecture and setup of the ELK Stack.

The solution provides:

- Centralized Log Collection
- Log Processing
- Log Storage
- Log Search
- Data Visualization

This project serves as the foundation for all future observability and monitoring projects.

---

# Business Scenario

Organizations generate logs from:

- Applications
- Servers
- Databases
- Security Systems
- Network Devices

Without centralized logging, troubleshooting and monitoring become difficult.

ELK Stack provides a unified platform for log management and analysis.
